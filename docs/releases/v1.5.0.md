# Release Notes - v1.5.0

> **Date**: 2025-12-15
> **Type**: Enhancement

## Summary

Mermaid 코드 보기 UX 개선 - 툴팁, 복사 버튼, 시각적 피드백 추가.

## Enhancements

### Mermaid 코드 보기 UX 개선

- **툴팁 추가**: 코드 보기 아이콘(`<>`)에 마우스를 올리면 "Mermaid 코드 보기" 표시
- **복사 버튼 추가**: 코드가 표시되면 우측 상단에 "복사" 버튼 표시
  - 클릭 시 전체 Mermaid 코드를 클립보드에 복사
  - 복사 성공 시 2초간 "복사됨" 상태 표시 (녹색 배경 + Check 아이콘)
- `navigator.clipboard.writeText()` API 사용
- `copiedFor` state 추가, 2초 후 자동으로 초기 상태 복귀
