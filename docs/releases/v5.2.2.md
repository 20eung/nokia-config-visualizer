# Release Notes - v5.2.2

> **Date**: 2026-03-03
> **Branch**: `main`
> **Type**: Bug Fix + UI (Patch Release)

## Summary

대시보드 Epipe 이중집계 버그 수정 및 Services UI 개선. Services 헤더 간소화 및 그룹 기본값 접힘으로 변경했습니다.

## Bug Fixes

### 대시보드 Epipe 이중집계 수정

**증상**: 대시보드 통계 카드의 Epipe 카운트가 Services 패널과 2배 차이

**원인**: Epipe는 두 사이트를 연결하는 P2P 서비스로 각 사이트에서 한 번씩 집계되어 2배 계산

**수정**: `siteGroups.serviceCounts` 합산 → `configs` 기반 고유 서비스 ID 집계

## UI/UX Improvements

### Services 헤더 개선

- "Network Services" → "Services"로 간소화
- 카운트 표시: 선택 항목 있을 때 선택 수, 없을 때 전체 서비스 수
- IES는 인터페이스 단위(= 고객 서비스 단위)로 정확하게 집계

### 그룹 기본값 접힘

- 초기 로딩 시 Epipe/VPLS/VPRN/IES 그룹 모두 접힌 상태로 시작
- 검색어 없을 때 그룹 강제 펼침 버그 수정

### 그룹 헤더 텍스트 간소화

- "VPLS Services" → "VPLS", "VPRN Services" → "VPRN", "IES Services" → "IES"
