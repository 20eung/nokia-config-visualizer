# Release Notes - v4.6.0

> **Date**: 2026-02-19
> **Branch**: `main`
> **Type**: Feature (Minor Release)

## Summary

IP 서브넷 검색 기능 추가. 검색창에서 IPv4 주소를 입력하면 자동으로 서브넷 검색 모드가 활성화되며 Longest Prefix Match(LPM) 알고리즘으로 가장 적합한 서브넷을 우선 표시합니다.

## New Features

### IP 서브넷 검색 (IP Subnet Search)

- IPv4 주소 입력 시 자동 IP 검색 모드 활성화
- Static Route 설정 기반 서브넷 매칭 (예: `10.0.1.50` 검색 → `10.0.1.0/24` 매칭)
- Longest Prefix Match (LPM) 알고리즘으로 가장 구체적인 서브넷 우선 정렬
- IES 서비스의 인터페이스 레벨 필터링 (검색 IP와 관련된 인터페이스만 표시)

### IPv4 유틸리티 모듈 (`src/utils/ipUtils.ts`)

- `isValidIPv4()` — IPv4 주소 유효성 검증
- `ipToLong()` — IP 주소를 32비트 정수로 변환
- `parseNetwork()` — CIDR 표기법 파싱
- `isIpInSubnet()` — IP 주소의 서브넷 포함 여부 확인
- `sortByLongestPrefix()` — Longest Prefix Match 정렬

### 검색 알고리즘

1. IPv4 정규식 검증 및 각 옥텟 0-255 범위 확인
2. 모든 서비스의 Static Routes에서 검색 IP 포함 서브넷 탐색
3. prefixLen 내림차순 정렬 (LPM)
4. IES 서비스: 관련 라우트를 가진 인터페이스만 필터링
5. `/8` 미만의 과도하게 넓은 서브넷 자동 제외
