# Release Notes - v1.7.0

> **Date**: 2025-12-15
> **Type**: Feature

## Summary

VRRP VIP 및 Master 표시 기능 추가. VRRP 설정이 있는 인터페이스의 VIP와 Master 여부를 다이어그램에 표시.

## New Features

### VRRP VIP 표시

- **VIP**: IP 주소 아래에 `(VIP: x.x.x.x)` 형식으로 표시
- **Master**: Priority >= 100인 경우 IP 앞에 `*` 표시

**예시**:
```
# Master (priority 110)
IP: *51.51.34.218/30
(VIP: 51.51.34.217)

# Backup (priority 90)
IP: 51.51.34.222/30
(VIP: 51.51.34.217)
```

## Technical Details
