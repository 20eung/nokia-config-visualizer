# Release Notes - v5.8.2

> **Date**: 2026-04-10
> **Branch**: `main`
> **Type**: UX Enhancement

## 변경 요약

Dashboard 모드에서 사이트 카드를 클릭해 Services 뷰로 전환할 때, 해당 장비의 서비스만 왼쪽 패널에 필터링하여 표시하는 기능을 추가했습니다.

## 주요 변경 사항

### feat: Dashboard 카드 → Services 장비 필터링

**이전 동작:**
- Dashboard 카드 클릭 → Services 전환 시 왼쪽 패널에 전체 장비 서비스 표시 (예: Epipe 0/348)
- 타입 버튼(All, Epipe 등) 클릭 시 전체 장비 서비스가 선택됨

**개선 동작:**
- Dashboard 카드 클릭 시 해당 장비의 서비스만 왼쪽 패널에 표시
- 사이드바 상단 파란 배너에 장비명 표시 + X 버튼으로 필터 즉시 해제 가능
- 헤더 Services 탭 직접 클릭 시 필터 초기화 (전체 서비스 표시)
- HA 페어 카드 클릭 시 두 장비의 서비스 합산 표시

**수정 파일:** `src/pages/V3Page.tsx` (단일 파일, ~30줄 추가)

## 기술 상세

- `dashboardFilterHostnames: string[]` 상태 추가
- `displayedServices` useMemo: `allServices.filter(s => hostnames.includes(s._hostname))`
- `allServices` 원본 보존 — 다이어그램 생성 로직 영향 없음
- `ServiceListV3` props interface 변경 없음 — `services` prop만 교체
