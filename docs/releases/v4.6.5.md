# Release Notes - v4.6.5

> **Date**: 2026-02-19
> **Branch**: `main`
> **Type**: Feature (Patch Release)

## Summary

검색창 아래에 6개의 클릭 가능한 예시 pill UI 추가. 사용자가 자주 쓰는 검색 패턴을 클릭 한 번으로 입력할 수 있습니다.

## New Features

### 검색 예시 UI (Search Examples Pills)

- 검색창 아래에 6개 클릭 가능한 예시 pill 표시
  - QoS, IP 서브넷, AND 검색, 서비스 ID, 포트, 서비스 타입 예시
- 클릭 시 검색창에 자동 입력 (사용자가 수정 가능)
- Tooltip으로 각 예시 설명 표시 (호버 시)
- 키보드 접근성 지원 (Tab 네비게이션, Enter/Space)

### UI/UX 개선

- Hover/Active 효과: 예시 pill에 시각적 피드백 (색상 변경, 애니메이션)
- 반응형 디자인: 모바일(< 768px) 및 태블릿(768-1024px) 최적화
- 접근성: aria-label, title 속성으로 스크린 리더 지원
