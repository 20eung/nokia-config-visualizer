# Release Notes - v4.8.1

> **Date**: 2026-02-23
> **Branch**: `main`
> **Type**: Bug Fix (Patch Release)

## Summary

원격 접속 환경에서 WebSocket 연결이 실패하던 문제 수정. 하드코딩된 `localhost:3001` 참조를 제거하여 원격 클라이언트와의 양방향 통신을 지원합니다.

**Nginx 설정**:
- `/ws` 경로에 WebSocket 프록시 설정 추가
- 원격 클라이언트와의 양방향 WebSocket 통신 지원

**프론트엔드**:
- 하드코딩된 `localhost:3001` 참조 제거
- `window.location.host` 및 상대 경로 사용으로 원격 접속 최적화
