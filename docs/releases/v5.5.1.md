# Release Notes - v5.5.1

> **Date**: 2026-03-16

## Summary

Copy PNG 버튼이 클립보드 복사 대신 파일 다운로드되는 문제 수정. NPM(nginx proxy manager) 등 리버스 프록시 환경에서 Permissions Policy 헤더 누락으로 인한 Clipboard API 차단 해결.

## Bug Fixes

### Copy PNG 클립보드 복사 기능 수정

**증상**: 다이어그램 패널의 'Copy PNG' 버튼이 클립보드에 이미지를 복사하는 대신 PNG 파일을 다운로드

**원인**: 리버스 프록시를 통한 HTTPS 접속 시 Permissions-Policy 헤더 누락으로 Clipboard API 차단

**수정 내용**:

1. **애플리케이션 레벨** (`src/components/v3/ServiceDiagram.tsx`):
   - Clipboard API 우선 시도, 실패 시 자동으로 다운로드 fallback
   - Permissions Policy로 차단된 경우 사용자 친화적 콘솔 메시지

2. **Docker nginx 설정** (`nginx.conf`):
   - 각 location 블록에 `Permissions-Policy: clipboard-write=*, clipboard-read=*` 헤더 추가

3. **NPM(nginx proxy manager) 설정 가이드**:
   - Advanced 탭 → Custom Nginx Configuration에 추가:
   ```nginx
   add_header Permissions-Policy "clipboard-write=(self), clipboard-read=(self)" always;
   ```
