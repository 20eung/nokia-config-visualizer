# Release Notes - v4.1.0

> **Type**: Feature
> **Full Changelog**: https://github.com/20eung/nokia-config-visualizer/compare/v4.0.0...v4.1.0

## Summary

AI 기반 이름 사전(Name Dictionary) 기능 추가. Config description에서 엔티티를 AI가 추출하여 한국어/별칭으로 서비스 검색 가능.

## New Features

### 이름 사전 (Name Dictionary)

- **AI 자동 생성**: Config description에서 고객명, 지역, 서비스명, 장비명 등 자동 추출
- **이름 변형 생성**: 짧은 이름, 긴 이름, 한국어 이름, 별칭 자동 생성
- **증분 구축**: Config 파일 점진적 업로드 시 사전 누적 구축 (기존 항목 보존, 새 토큰만 추가)
- **AI 챗봇 연동**: 한국어/별칭으로 질문해도 원본 토큰과 매칭하여 서비스 검색
- **전역 단일 사전**: `/app/data/dictionary.json` 단일 파일 저장, Docker named volume으로 재빌드 시 데이터 유지

### Dictionary Editor UI

- 모달 기반 사전 편집기 (이름 사전 버튼으로 접근)
- 카테고리별 분류: customer, location, service, device, other
- 컬럼 헤더 클릭 정렬 (한국어 로케일 지원)
- 중복 정리, 전체 삭제 기능

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/dictionary/generate` | AI 사전 자동 생성 |
| GET | `/api/dictionary` | 전역 사전 로드 |
| PUT | `/api/dictionary` | 전역 사전 저장 |

- AI 생성 중복 데이터: originalToken이 shortName/longName/koreanName/aliases에 복사되던 문제 수정
