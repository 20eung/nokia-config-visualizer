# Release Notes - v5.4.0

> **Date**: 2026-03-11
> **Branch**: `main`
> **Type**: Feature (Minor Release)

## Summary

v4 AI Platform 기능을 v5 베이스로 성공적으로 이식한 통합 버전. NCV REST API, MCP Server, ConfigStore, useConfigSync를 v5 UI/보안 개선사항과 함께 통합합니다.

## New Features

### 백엔드 AI Platform 통합

**NCV REST API** (`server/src/routes/ncv.ts`) — 8개 엔드포인트:
- `POST /api/ncv/analyze` — Config 파싱 결과 백엔드 동기화
- `GET /api/ncv/services` — 서비스 목록 조회 (type/hostname/query 필터)
- `GET /api/ncv/services/:key` — 서비스 상세 정보 조회
- `GET /api/ncv/topology` — 네트워크 토폴로지 생성 (JSON/Mermaid)
- `GET /api/ncv/devices` — 장비 목록 조회
- `GET /api/ncv/search` — 키워드 검색
- `GET /api/ncv/export` — 전체 데이터 내보내기
- `GET /api/ncv/stats` — 시스템 통계

모든 NCV 엔드포인트는 `X-API-Key` 헤더 필요 (v5 보안 정책 준수)

**ConfigStore** (`server/src/services/configStore.ts`): 싱글톤 기반 In-Memory 파싱 결과 저장소

**MCP Server** (`server/src/mcp-server.ts`):
- HTTP MCP 엔드포인트 (`POST/GET /mcp`)
- 독립 실행 서버 (stdio)
- 7개 MCP 도구: get_services, get_service_detail, get_topology, search_config, get_devices, get_stats, get_ha_pairs

### 프론트엔드 통합

**useConfigSync 훅** (`src/hooks/useConfigSync.ts`):
- Config 로드 시 백엔드 ConfigStore 자동 동기화
- `V3Page.tsx`에서 `useConfigSync(configs)` 1줄로 활성화
- Demo 모드 또는 백엔드 미연결 시 조용히 무시
