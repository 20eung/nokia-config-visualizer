# Release Notes - v1.8.0

> **Date**: 2025-12-17
> **Type**: Feature + Security

## Summary

VRRP 기반 HA 탐지 기능 추가 및 Config 파일 보안 개선 (민감 정보 익명화).

## New Features

### VRRP 기반 HA 자동 탐지

- **VRRP VIP 기반 HA Pair 인식**: 동일한 VRRP Virtual IP를 공유하는 인터페이스들을 자동으로 이중화 쌍으로 탐지
- **향상된 HA 필터**: Static Route 기반 이중화뿐만 아니라 VRRP 기반 이중화도 "이중화" 필터에서 자동 선택
- **완전한 Peer IP 추론**: /30 서브넷뿐만 아니라 VRRP가 설정된 모든 서브넷에서 peer IP 자동 추론
- VRRP VIP별 인터페이스 그룹화 로직 추가
- 기존 Static Route 기반 HA 탐지 기능 완전 유지

## Security

- Config 파일의 모든 민감 정보 익명화 완료
- 실제 회사명 → Customer1~10으로 변경
- 공인 IP → 사설 IP (10.x.x.x) 변경
- 공개 저장소 데모용으로 안전하게 정리
