# Release Notes - v5.7.4

> **Date**: 2026-04-10
> **Branch**: `main`
> **Type**: Enhancement

## 변경 사항

### Mermaid 다이어그램 렌더링 방식 전면 개편

`src/utils/v3/mermaidGeneratorV3.ts` 전체 리팩토링.

#### 주요 변경

**1. flowchart LR 전환**
- `graph LR` → `flowchart LR` (Epipe / VPLS / VPRN / IES / Full topology 전체)
- Mermaid 최신 권장 문법으로 통일

**2. classDef 구조 개선**
- `classDef default` 제거 → `classDef hostNode` 신규 (stroke-width:1px, text-align:left)
- `classDef service/vpls` → `classDef serviceNode` (text-align:left 추가)
- class 기반 텍스트 정렬로 `<div style='text-align: left'>` 래퍼 제거

**3. 노드 레이블 포맷 변경**
- `<div style='text-align: left'>` 래퍼 제거 (classDef로 대체)
- Unicode 이스케이프 제거 (`\u003cb\u003e` → `<b>`)
- Level 1 bullet: `‑ ` → `- `
- Level 2 bullet (Ethernet 하위): `  ‑ ` → `&nbsp;&nbsp;&nbsp;- `

**4. 들여쓰기 구조 적용**
- 최상위 구문: 4-space 들여쓰기
- subgraph 내부 노드: 8-space 들여쓰기
- 섹션 구분 주석(`%% ...`) 추가

**5. class 할당 방식 변경**
- 인라인 할당 제거 → 최하단 일괄 할당
- `class SAP1,SAP2 hostNode` / `class SVC serviceNode` 형식

**6. 화살표 길이 증가**
- `-->` → `--->` (렌더링 엔진이 노드 간격을 넓히도록)

**7. subgraph Hostname 줄바꿈 방지**
- subgraph 레이블에 `noWrap()` 적용
- `-` → `\u2011` (non-breaking hyphen) 변환으로 Hostname 줄바꿈 완전 차단
