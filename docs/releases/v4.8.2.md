# Release Notes - v4.8.2

> **Date**: 2026-02-24
> **Branch**: `main`
> **Type**: Bug Fix (Patch Release)

## Summary

HTTP 환경에서 Clipboard API의 Secure Context 제한으로 Copy PNG가 실패하던 문제 수정. Clipboard API 실패 시 PNG 파일 자동 다운로드로 fallback 처리.

## Bug Fixes

### Copy PNG Fallback

**증상**: HTTP 환경에서 'Copy PNG' 버튼 클릭 시 오류 발생 (아무것도 안 됨)

**원인**: `navigator.clipboard.write()` API가 Secure Context(HTTPS)에서만 동작

**수정**: Clipboard API 실패 시 PNG 파일 자동 다운로드로 fallback 처리

- HTTPS 환경: Clipboard API → 클립보드 복사
- HTTP 환경: Clipboard API 실패 → PNG 파일 다운로드
