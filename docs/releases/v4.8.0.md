# Release Notes - v4.8.0

> **Date**: 2026-02-21
> **Branch**: `main`
> **Type**: Feature (Minor Release)

## Summary

Nokia Config Visualizer를 AI 협업 미들웨어로 확장하는 NCV AI Collaboration Platform 기능을 추가했습니다. NCV REST API, MCP Server, 시맨틱 검색(RAG) 세 가지 주요 기능을 포함합니다.

## New Features

### Feature 1: Structured JSON Output API

**NCV REST API** (`server/src/routes/ncv.ts`) — 8개 엔드포인트:
- `POST /api/ncv/analyze` — 프론트엔드 파싱 결과 백엔드 동기화
- `GET /api/ncv/services` — 서비스 목록 (type/hostname/q 필터)
- `GET /api/ncv/services/:serviceKey` — 서비스 상세 조회
- `GET /api/ncv/topology` — 토폴로지 (JSON/Mermaid)
- `GET /api/ncv/devices` — 장비 목록
- `GET /api/ncv/search` — 키워드 검색
- `GET /api/ncv/export` — 전체 데이터 내보내기
- `GET /api/ncv/stats` — 시스템 통계

**ConfigStore** (`server/src/services/configStore.ts`): 싱글톤 기반 In-Memory 파싱 결과 저장소

**useConfigSync** (`src/hooks/useConfigSync.ts`): Config 로드 시 백엔드 자동 동기화 React 훅

### Feature 2: MCP Server (Model Context Protocol)

- **stdio MCP 서버** (`server/src/mcp-server.ts`): Claude Desktop, Cursor 직접 연결
- **HTTP MCP 엔드포인트** (`POST/GET /mcp`): 웹 기반 AI 에이전트용
- **7개 MCP 도구**: get_services, get_service_detail, get_topology, search_config, get_devices, get_stats, get_ha_pairs

### Feature 3: 시맨틱 검색 (RAG Indexing)

- **chunkBuilder.ts**: 서비스 데이터 → 자연어 텍스트 청크 변환
- **embeddingService.ts**: Amazon Titan Embed Text v2 (Bedrock) 기반 벡터 생성
- **ragIndexer.ts**: vectra 로컬 파일 기반 벡터 DB
- **RAG 엔드포인트**: `/api/ncv/index`, `/api/ncv/semantic-search`, `/api/ncv/index/status`
