---

## 🐛 버그 수정

### Unicode 하이픈 문자 검색 오류 수정

#### 문제 상황
- **증상**: `SK‑Net_Gyowon_7705SAR‑8_MPLS` 같은 hostname을 `sk-net` 또는 `gyowon`으로 검색 불가
- **원인**: hostname에 Unicode Non-Breaking Hyphen (`U+2011`) 사용, JavaScript의 `.toLowerCase().includes()`는 Unicode 정규화 미지원
- **재현 조건**:
  ```javascript
  const hostname = 'SK‑Net_Gyowon_7705SAR‑8_MPLS'; // U+2011 포함
  const query = 'sk-net';                          // ASCII hyphen
  hostname.toLowerCase().includes(query.toLowerCase()); // false (실패)
  ```

#### 해결 방법

**1. Unicode 정규화 함수 추가** ([ServiceListV3.tsx:55-65](../src/components/v3/ServiceListV3.tsx#L55-L65))

```typescript
function normalizeSearchString(str: string): string {
  return str
    .normalize('NFKD')  // 호환 문자 분해
    .replace(/[\u2010-\u2015\u2212\uFE58\uFE63\uFF0D]/g, '-')  // 모든 하이픈 → ASCII
    .toLowerCase();
}
```

**지원 하이픈 문자** (7종):
| Unicode | 이름 | 예시 |
|---------|------|------|
| U+2010 | Hyphen | `‐` |
| U+2011 | Non-Breaking Hyphen | `‑` ← 문제 원인 |
| U+2012 | Figure Dash | `‒` |
| U+2013 | En Dash | `–` |
| U+2014 | Em Dash | `—` |
| U+2015 | Horizontal Bar | `―` |
| U+2212 | Minus Sign | `−` |
| U+FE58 | Small Em Dash | `﹘` |
| U+FE63 | Small Hyphen-Minus | `﹣` |
| U+FF0D | Fullwidth Hyphen-Minus | `－` |
