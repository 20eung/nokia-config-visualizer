# Release Notes - v3.1.0

> **Type**: Feature + Bug Fix
> **Full Changelog**: https://github.com/20eung/nokia-config-visualizer/compare/v3.0.0...v3.1.0

## Summary

VPRN 서비스 BGP/OSPF 시각화 강화 및 UI 개선.

## New Features

### Advanced VPRN Visualization

- **BGP/OSPF 통합 표시**: BGP 및 OSPF 라우팅 상세 정보를 서비스 노드 라벨 내에 통합 시각화
- **BGP Detail Box**: AS Number & Route Distinguisher (RD), Router ID, Neighbor 목록 (AS 번호 포함)
- **OSPF Detail Box**: OSPF Areas, 연결된 Interfaces (타입 표시)
- **중복 제거**: 서비스 요약에서 AS/RD 정보 중복 표시 제거

### UI/UX 개선

- 초기 화면 단순화 ("No Configuration Loaded" 메시지 개선)
- BGP/OSPF 섹션에 레드 스타일 적용 (네트워크 다이어그램 관례 준수)

## Bug Fixes

- 파서 내 중복 변수 선언으로 인한 빌드 실패 수정
- 서비스 라벨에서 AS/RD 정보 이중 표시 수정

## Technical Updates

- `parserV3.ts`: BGP/OSPF 설정 심층 추출 지원
- `mermaidGeneratorV3.ts`: 서비스 노드 임베디드 라벨 레이아웃 리팩터링
