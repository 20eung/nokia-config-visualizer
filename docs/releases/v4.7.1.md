# Release Notes - v4.7.1

> **Date**: 2026-02-20
> **Branch**: `main`
> **Type**: Bug Fix (Patch Release)

## Summary

Cloudflare Pages 등 정적 호스팅 환경에서 불필요한 WebSocket 연결 시도를 방지. Demo/Beta 환경에서는 수동 Upload만 사용 가능하도록 변경.

## Bug Fixes

### Demo 환경 WebSocket 연결 방지

**증상**: Cloudflare Pages 같은 정적 사이트에서 WebSocket 연결 시도 → 5번 재시도 → 콘솔 오류

**수정**: Demo/Beta/Cloudflare 환경 감지 시 WebSocket 연결 비활성화

감지 조건:
- `window.location.hostname.includes('demo')`
- `window.location.hostname.includes('beta')`
- `window.location.hostname.includes('pages.dev')`
- `window.location.hostname.includes('cloudflare')`

**변경 내용**:
- `useConfigWebSocket.ts`: Demo 환경에서 연결 시도 안 함 (status: 'disconnected' 유지)
- `ConfigFileList.tsx`: Demo 환경에서 연결 상태 badge/text 미표시
- `FolderPathSettings.tsx`: Demo 환경에서 기능 제한 경고 메시지 표시
