# Release Notes - v5.2.1

> **Date**: 2026-03-02
> **Branch**: `main`
> **Type**: UI/UX (Patch Release)

## Summary

Type 필터 버튼에 전체선택/해제 토글 기능을 통합하여 클릭 수를 절반으로 줄였습니다. 별도의 Select 영역을 제거합니다.

## UI/UX Improvements

### Type 버튼 통합 선택/해제

- `Select: 전체 / 해제` 버튼 영역 완전 제거
- Type 필터 버튼이 전체선택/해제 토글 기능 통합

**동작 방식**:
- **1회 클릭 (다른 타입)**: 필터 전환 + 전체 선택 → 다이어그램 즉시 표시
- **재클릭 (같은 타입, 선택 있음)**: 전체 해제 → 다이어그램 사라짐
- **재클릭 (같은 타입, 선택 없음)**: 전체 재선택 → 다이어그램 표시

클릭 수 감소: 타입 변경 + 전체선택 2번 → 1번으로 단축

`pendingSelectAll` ref 패턴으로 필터 변경 후 자동선택 구현
