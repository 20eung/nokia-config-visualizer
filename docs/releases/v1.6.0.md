# Release Notes - v1.6.0

> **Date**: 2025-12-15
> **Type**: Feature + Enhancement

## Summary

인터페이스 리스트 계층 구조(장비별 접기/펼치기) 추가 및 이중화 버튼 HA 감지 개선.

## New Features

### 인터페이스 리스트 계층 구조

파일 탐색기처럼 장비별로 인터페이스를 접고 펼칠 수 있습니다.

- 장비 헤더 클릭 시 인터페이스 리스트 토글
- ChevronRight(▶) 접힘 / ChevronDown(▼) 펼침
- 각 장비의 인터페이스 개수 표시
- 기본값: 모든 장비 펼쳐진 상태

## Enhancements

### 이중화 버튼 기능 개선

- mermaidGenerator와 동일한 로직 적용
- relatedRoutes 비교하여 공통 Customer Network 감지
- config 문법이 달라도 정확하게 작동

### UI 개선

- Chevron/Server 아이콘 크기 14px → 16px
- 아이콘 간격 4px → 6px
- `flexShrink: 0` 추가로 아이콘 찌그러짐 방지
