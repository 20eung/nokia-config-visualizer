# v1.5.0 - Mermaid Code Viewer UX Improvements

## 🎨 주요 개선사항

### Mermaid 코드 보기 UX 개선

사용자가 Mermaid 코드를 더 쉽게 확인하고 복사할 수 있도록 개선했습니다.

**1. 툴팁 추가**
- 코드 보기 아이콘(`<>`)에 마우스를 올리면 **"Mermaid 코드 보기"** 표시
- 사용자가 기능을 쉽게 인지할 수 있음

**2. 복사 버튼 추가**
- 코드가 표시되면 우측 상단에 **"복사" 버튼** 표시
- 클릭 시 전체 Mermaid 코드를 클립보드에 복사
- 복사 성공 시 2초간 **"복사됨"** 상태 표시
  - 녹색 배경으로 시각적 피드백
  - Check 아이콘으로 성공 표시

**3. 시각적 피드백**
- Copy 아이콘: 복사 전
- Check 아이콘: 복사 완료
- 색상 변화로 명확한 상태 표시

## 💡 사용 방법

1. 다이어그램 우측 상단의 `<>` 아이콘 클릭
2. Mermaid 코드 확인
3. 우측 상단의 "복사" 버튼 클릭
4. 클립보드에 전체 코드 복사됨
5. 2초간 "복사됨" 상태 표시

## 🔧 기술적 개선

- `navigator.clipboard.writeText()` API 사용
- 복사 상태 관리를 위한 `copiedFor` state 추가
- 2초 후 자동으로 초기 상태로 복귀
- Copy/Check 아이콘으로 시각적 피드백 제공
